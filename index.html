Introdução a orientação a objetos

Quando começamos a programar, geralmente aprendemos a criar códigos-fonte estruturados, que possuem começo, meio e fim. Com o passar do tempo, reparamos que sistemas mais complexos resultam em códigos-fonte gigantescos, compostos por milhares de linhas de código, que se tornam o pesadelo para manter funcionando: além da manutenção complicada, são difíceis de se reaproveitar trechos decódigo que inevitavelmente se repetirão muito. Para estes e outros problemas que existe a programação orientada a objetos (OOP), um paradigma totalmente diferente para planejar e organizar o código-fonte. Contudo, o primeiro contato com esta nova abordagem pode se tornar confuso e um tanto desencorajador. Caso você se sinta assim (ou queira evitar este sentimento), esse curso é para você. Nele, aprenderemos como funciona o paradigma orientado a objetos, realizando exemplos práticos de desenvolvimento de aplicações utilizado como ferramenta a linguagem Java. Venha aprender conosco uma forma de programação que será fundamental para seu desenvolvimento pessoal.
